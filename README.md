Developed a time series forecasting model for U.S. retail sales using Holt-Winters, capturing trend, seasonality, and economic shocks to improve prediction accuracy.

# Retail Sales Time Series Forecasting

## Overview
This project analyzes U.S. monthly retail sales data and builds a forecasting model using Holt-Winters exponential smoothing.

## Objective
To predict retail sales for the final 12 months and evaluate model performance using RMSE.

## Methods
- Data cleaning and reshaping
- Time series visualization
- Holt-Winters model (trend + seasonality)
- Model evaluation using RMSE

## Results
- Holt-Winters significantly outperformed seasonal naive baseline
- Successfully captured trend, seasonality, and COVID-19 impact

## Business Value
Supports demand forecasting, inventory planning, and retail decision-making.
